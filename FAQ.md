# FAQ

## What is demo-day-timer?
Presentation timer with cue cards. Practice mode tracks pacing. Generates speaker notes from deck content.

## How do I install it?
```bash
git clone https://github.com/josezuma/demo-day-timer.git && cd demo-day-timer
```

## Is this compatible with yc-pitch-deck-max?
Yes. demo-day-timer integrates directly with yc-pitch-deck-max.

## Do I need any external dependencies?
No. All scripts use only Python stdlib.
